# Integrator
Application dividing employees into groups to start some relationships between them.

# Technologies
## Frontend
- React.js
  - ES6 syntax
  - JSX
- CSS
  - Styled Components
- NPM

## Backend
- Node.js
  - Express.js
- PostgreSQL
  - Python - to create some powering queries

## Managing
- Docker
  - Alpine-based node images
- Docker-compose
- Make

# Running
 Simply run *make* to first run the project.

## Build
- make build

## Run
- make run

## Stop
- make stop
